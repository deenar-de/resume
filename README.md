## 開發 / Development

```bash
npm install
npm run dev      # 本地預覽 http://localhost:3030
npm run build    # 產生靜態網站到 dist/
npm run export   # 匯出 PDF
```

## 部署 / Deploy

推送到 `main` 分支後，GitHub Actions（`.github/workflows/deploy.yml`）會自動 build 並發佈到 GitHub Pages。
Pushing to `main` triggers GitHub Actions to build and publish to GitHub Pages.
